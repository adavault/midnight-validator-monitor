## Summary

Security patch updating two transitive dependencies with known vulnerabilities flagged by `cargo audit`.

---

## Security Fixes

### RUSTSEC-2026-0007 — `bytes` 1.11.0 → 1.11.1

**Severity:** Memory corruption
**CVE:** CVE-2026-25541

Integer overflow in `BytesMut::reserve` could cause out-of-bounds writes in release builds. The overflow in `new_cap + offset` can wrap `usize`, corrupting the internal capacity and enabling unsafe memory access.

### RUSTSEC-2026-0009 — `time` 0.3.45 → 0.3.47

**Severity:** Denial of service
**CVE:** CVE-2026-25727

Stack exhaustion via malicious RFC 2822 input. Recursive parsing of deprecated RFC 2822 comment nesting could exhaust the stack with crafted input.
